# 🍽️ Restaurant Analytics Dashboard | Power BI

## 📌 Project Overview

The **Restaurant Analytics Dashboard** is an interactive Power BI project designed to analyze restaurant performance, sales trends, customer behavior, and operational metrics. The dashboard transforms raw restaurant data into actionable insights using dynamic visualizations, KPIs, and business intelligence techniques.

This project demonstrates practical skills in **Power BI**, **Power Query**, **DAX**, **data modeling**, and **interactive dashboard design**.

---

## 🎯 Objectives

* Analyze restaurant sales performance.
* Monitor revenue and order trends.
* Identify top-performing menu items.
* Understand customer purchasing behavior.
* Track operational performance through key metrics.
* Support data-driven business decisions.

---

## 📊 Dashboard Features

* 💰 Total Revenue KPI
* 🧾 Total Orders
* 📈 Sales Trend Analysis
* 🍕 Top-Selling Menu Items
* 🥗 Category-wise Sales
* 👥 Customer Insights
* 📅 Daily & Monthly Sales Analysis
* 🏪 Branch/Location Performance (if applicable)
* 🎛️ Interactive Filters and Slicers
